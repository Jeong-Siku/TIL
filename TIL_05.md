## 딕셔너리
>키와 값이 쌍으로 구성된다. 인덱스 대신에 키를 통해 값을 다룬다.
- {key1:value1,key2:value2....}
- 변수명[키]
- `데이터`에는 숫자나 문자열 , `값`에는 문자열,리스트,튜플,딕셔너리,숫자 입력 가능
- 데이터 추가 : `dict_variable[key]=value`
- 데이터 변경 : `dict_variable[key]=value`
- 데이터 삭제 : `del dict_variable[key]`
